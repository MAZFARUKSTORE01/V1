

## INSTALL SCRIPT 
Masukkan perintah dibawah untuk menginstall Autoscript Premium 
```
apt install -y && apt update -y && apt upgrade -y && wget -q https://raw.githubusercontent.com/MAZFARUKSTORE01/V1/main/instal.sh && chmod +x instal.sh && ./instal.sh
```
